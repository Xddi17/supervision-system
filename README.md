# 办公室管理系统

第一期已搭好局域网页面与本地数据存储框架，当前重点模块为制度库和后勤管理。

## 当前能力

- 主管理员登录、邀请码生成、员工注册、注册审核
- 制度库上传、分类、版本、状态、文件查看
- 后勤管理骨架
  - 办公用品：请购单、采购单、入库单、出库单台账
  - 租赁：合同上传与字段预留
  - 办公室分配：平面图上传、房间名称与使用人维护
  - 固定资产：资产台账、移交/报废申请与审批
- 会议室、督办、三会、人事、宣传模块预留入口
- 统一 AI 窗口预留

## 本地数据目录

程序启动后会自动在桌面创建：

`/Users/xdd/Desktop/办公室管理系统`

其中包括：

- `data/db.json`：本地数据库
- `attachments/policies`：制度文件
- `attachments/contracts`：合同文件
- `attachments/floorplans`：平面图
- `attachments/assets`：资产附件预留
- `attachments/import-export`：导入导出预留

## 启动方式

```bash
npm start
```

默认访问地址：

`http://localhost:3000`

局域网访问时，用本机 IP 加端口访问，例如：

`http://192.168.x.x:3000`
